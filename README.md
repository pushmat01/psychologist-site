# Сайт психолога-консультанта

Статический сайт-визитка на HTML, CSS и JavaScript. Его можно бесплатно опубликовать на GitHub Pages, Vercel или Netlify.

## Структура проекта

- `index.html` - разметка сайта и тексты.
- `styles.css` - визуальный стиль и адаптивность.
- `script.js` - мобильное меню.
- `README.md` - инструкция по запуску и публикации.

## Как открыть сайт локально

1. Скачайте или откройте папку проекта на компьютере.
2. Дважды нажмите на файл `index.html`.
3. Сайт откроется в браузере.

Можно также запустить локальный сервер:

```bash
python3 -m http.server 8000
```

После этого откройте в браузере:

```text
http://localhost:8000
```

## Как заменить тексты, фото и контакты

Откройте `index.html` в любом редакторе кода или обычном текстовом редакторе.

Что важно заменить:

1. Имя или название личного бренда в шапке сайта.
2. Текст в блоке "Обо мне".
3. Заглушку "Ваше фото" на свое фото.
4. Стоимость консультации в блоке "Формат и стоимость".
5. Все ссылки `https://t.me/username` на свою ссылку в Telegram.
6. Текст в подвале сайта.

В коде уже есть комментарии рядом с местами, которые нужно изменить.

### Как добавить фото

1. Положите файл фото в папку проекта, например `photo.jpg`.
2. В `index.html` найдите блок:

```html
<div class="photo-placeholder">
  <span>Ваше фото</span>
</div>
```

3. Замените его на:

```html
<img class="profile-photo" src="photo.jpg" alt="Ваше имя, психолог-консультант">
```

Стили для класса `profile-photo` уже добавлены в `styles.css`.

## Как загрузить проект на GitHub

1. Зарегистрируйтесь или войдите на [github.com](https://github.com).
2. Нажмите `New repository`.
3. Введите название репозитория, например `psychologist-site`.
4. Выберите `Public`.
5. Нажмите `Create repository`.
6. На странице репозитория нажмите `uploading an existing file`.
7. Перетащите файлы `index.html`, `styles.css`, `script.js`, `README.md`.
8. Нажмите `Commit changes`.

## Как бесплатно опубликовать через GitHub Pages

1. Откройте репозиторий на GitHub.
2. Нажмите `Settings`.
3. В левом меню выберите `Pages`.
4. В разделе `Build and deployment` выберите `Deploy from a branch`.
5. В поле `Branch` выберите `main` и папку `/root`.
6. Нажмите `Save`.
7. Подождите 1-3 минуты.
8. Готовая ссылка появится в разделе `Pages`.

Обычно ссылка выглядит так:

```text
https://ваш-логин.github.io/psychologist-site/
```

## Как бесплатно опубликовать через Vercel

1. Зайдите на [vercel.com](https://vercel.com).
2. Нажмите `Sign Up` или `Log In`.
3. Выберите вход через GitHub.
4. Нажмите `Add New...` → `Project`.
5. Выберите репозиторий с сайтом.
6. В настройках проекта ничего менять не нужно.
7. Нажмите `Deploy`.
8. Через несколько минут Vercel выдаст ссылку на сайт.

## Как бесплатно опубликовать через Netlify

1. Зайдите на [netlify.com](https://netlify.com).
2. Нажмите `Sign up` или `Log in`.
3. Выберите вход через GitHub.
4. Нажмите `Add new site` → `Import an existing project`.
5. Выберите GitHub и репозиторий с сайтом.
6. В поле `Build command` ничего не указывайте.
7. В поле `Publish directory` укажите:

```text
/
```

8. Нажмите `Deploy site`.
9. После публикации Netlify покажет ссылку на сайт.

## Перед публикацией

Проверьте:

- Telegram-ссылки заменены на ваши.
- Стоимость указана или оставлена как "указать позже".
- Текст "Обо мне" звучит естественно и честно.
- Фото, если добавлено, открывается на сайте.
- Сайт хорошо выглядит на телефоне и компьютере.
